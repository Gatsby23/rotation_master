# Rotation Master
- Provide conversion between the following representations of 3D rotation and display the pose
    - Rotation Matrix
    - Rotation Vector
    - Axis-angle
    - Quaternion
    - Euler Angle (Intrinsic(Mobile))
    - Euler Angle (Extrinsic(Fixed))
