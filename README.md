# Rotation Master
- Provide conversion between the following representations of 3D rotation (rigid body orientation) and display the pose
    - Rotation Matrix
    - Rotation Vector
    - Axis-angle
    - Quaternion
    - Euler Angle (Intrinsic; Mobile)
    - Euler Angle (Extrinsic; Fixed)
